# 开发这个插件的背景
当前市面上的Google 翻译插件不太好用，像有道云翻译插件等等；感觉不太适合程序员使用，因为有些技术文档经过翻译之后反而没那么易懂了，有些专业术语不翻译可能更方便理解。
例：“The event loop is responsible for handling asynchronous callbacks.”
有道云翻译：“事件循环负责处理异步回调。”
我想要的翻译：“event loop 会在 next tick 处理 callbacks”

# 谷歌插件描述
一个面向程序员的 Chrome 插件，将英文技术内容转换为“最小认知成本的中英混合表达”。

# UI交互
用户可通过鼠标右键选择进行全文翻译，也可以进行选中翻译；翻译内容直接在原文下方展示。
